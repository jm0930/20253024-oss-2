## 1. top이란?

`top`은 리눅스의 “작업 관리자” 같은 명령어입니다.  
실행 중인 프로그램들이 CPU나 메모리를 얼마나 쓰는지 **실시간으로 보여줍니다.**

---

## 사용방법

top은 터미널에서 아래 명령어를 입력하면 실행됩니다.



실행하면 화면이 자동으로 업데이트되며  
CPU나 메모리를 많이 사용하는 프로그램이 위쪽에 표시됩니다.

---

## 화면에서 보이는 정보

- CPU 사용률  
- 메모리 사용량  
- 실행 중인 프로세스 목록  
- 프로세스 ID(PID)  

위쪽은 시스템 전체 요약,  
아래쪽은 프로그램 목록이라고 보면 됩니다.

---

## 자주 쓰는 키

| 키 | 기능 |
|----|------|
| P | CPU 사용량 기준 정렬 |
| M | 메모리 사용량 기준 정렬 |
| k | 프로세스 종료(PID 입력) |
| q | top 종료 |

---

## 간단 예시

- CPU 많이 쓰는 프로그램 보기  
top → P
- 메모리 많이 쓰는 프로그램 보기  
top → M
- 특정 프로그램 종료  
top → k → PID 입력

## 2. ps란?

`ps`는 현재 실행 중인 프로세스를 **한 번에 출력해주는 명령어**입니다.  
top처럼 실시간은 아니고, **딱 실행한 순간의 상태(Snapshot)** 을 보여줍니다.

---

## 사용법

기본 사용은 아래처럼 입력합니다.
ps

조금 더 자세히 보고 싶으면 이렇게 많이 씁니다:


ps -e
ps aux

---

## 화면에서 보이는 정보

- 실행 중인 프로세스 목록  
- PID(프로세스 ID)  
- USER(누가 실행했는지)  
- CPU/메모리를 얼마나 쓰는지  
- COMMAND(실행 중인 프로그램 이름)  

---

## 자주 쓰는 옵션

| 옵션 | 설명 |
|------|------|
| -e | 시스템 전체 프로세스 보기 |
| aux | 모든 사용자 + 자세한 정보 표시 |
| -f | full format(계층/부모 PID까지 표시) |

---

## 간단 예시

- 전체 프로세스 보기  
ps -e
- 자세히 보기 (가장 많이 사용함)  
ps aux
- 특정 프로그램 찾기 (grep과 함께 사용)  
ps aux | grep python
