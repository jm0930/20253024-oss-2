## top이란?

`top`은 리눅스의 “작업 관리자” 같은 명령어입니다.  
실행 중인 프로그램들이 CPU나 메모리를 얼마나 쓰는지 **실시간으로 보여줍니다.**

---

## 사용방법

top은 터미널에서 아래 명령어를 입력하면 실행됩니다.



실행하면 화면이 자동으로 업데이트되며  
CPU나 메모리를 많이 사용하는 프로그램이 위쪽에 표시됩니다.

---

## 화면에서 보이는 정보

- CPU 사용률  
- 메모리 사용량  
- 실행 중인 프로세스 목록  
- 프로세스 ID(PID)  

위쪽은 시스템 전체 요약,  
아래쪽은 프로그램 목록이라고 보면 됩니다.

---

## 자주 쓰는 키

| 키 | 기능 |
|----|------|
| P | CPU 사용량 기준 정렬 |
| M | 메모리 사용량 기준 정렬 |
| k | 프로세스 종료(PID 입력) |
| q | top 종료 |

---

## 간단 예시

- CPU 많이 쓰는 프로그램 보기  
top → P
- 메모리 많이 쓰는 프로그램 보기  
top → M
- 특정 프로그램 종료  
top → k → PID 입력
